폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form태그 내의 input 태그들을 통해 사용자가 입력한 정보를 **서버**로 넘기는 역할을 한다.

* action 속성 : 폼에 입력된 값들을 전송받을 **클래스(java) 이름**을 입력한다.
* method 속성 : **get/post** 방식으로 전송할 방식을 지정한다.
** get 방식 : url주소에 form태그를 통해 넘어가는 데이터가 보인다.**
** post 방식 : url주소에 form태그를 통해 넘어가는 데이터가 보이지 않는다. **

 

submit 버튼을 눌렀을 때 action에 지정한 경로로 method에 지정한 방식에 따라서
input태그에 입력한 값들을 서버로 전달한다.
